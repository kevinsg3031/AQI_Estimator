The problem

India's CPCB monitors air quality in major cities, but hundreds of smaller towns have no monitoring stations. Standard ML approaches train a separate model per city — which requires historical data for each city, defeating the purpose.

The approach

This project uses Option B: pooled regression with location features — a single model trained on 13 cities that learns the relationship between a city's structural characteristics (industrial load, population density, region, coastal proximity) and its AQI. That learned relationship is then applied to cities the model has never seen.

Why city-level holdout matters

A random 80/20 train-test split would let the model see 95% of Delhi's data in training and test on the remaining 5% — that measures memorisation, not generalisation. Holding out entire cities forces the model to predict from structure alone, which is the actual problem being solved.
